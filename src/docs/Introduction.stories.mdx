# Welcome to Storybook do Twitter Web

Storybook helps you build UI components in isolation from your app's business logic, data, and context.
That makes it easy to develop hard-to-reach states. Save these UI states as **stories** to revisit during development, testing, or QA.

Para desenvolver o projeto, foram utilizadas as seguintes ferramentas:
TypeScript
React
CSS
Storybook
Node.JS
Tailwind CSS
Vite
React Router Dom
Figma
Google Fonts

O Twitter Web possui o modo do tema claro e escuro e permite postar um tweet.

Veja o `Layout` do projeto.
We recommend building UIs with a [**Compass**](https://www.figma.com/file/trDL8CBzu7wOxuyr7F3SDp/UI-Twitter-Web-(Community)-(Copy)?node-id=0%3A1).
